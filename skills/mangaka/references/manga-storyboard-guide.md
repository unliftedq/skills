# Manga Script and `Name` Storyboard Guide

Use this reference when turning a treatment into a manga script and rough `name` storyboard.

## Story Shape

Manga pages are not prose with pictures. Build the chapter around visual beats:

- **Hook**: an opening image, question, conflict, or mood that makes the reader turn the page or keep scrolling.
- **Escalation**: each page changes the situation, reveals information, or intensifies emotion.
- **Turn**: the decision, reversal, joke, discovery, or attack that changes the scene's direction.
- **Release**: a final image, quiet beat, punchline, cliffhanger, or emotional landing.

For short works, keep the cast small and make one clear emotional promise.

## Page Rhythm

Vary page density intentionally:

- 1 panel: impact, reveal, splash, place-setting, emotional silence.
- 2-3 panels: slower dramatic focus or clear action beats.
- 4-6 panels: standard scene progression.
- 7-9 panels: fast chatter, comedy, tactical action, or constrained tension.
- 10+ panels: use rarely; readability is fragile.

End pages on a turn, reveal, question, or image with enough pull to justify the next page.

## Single-Page Layout Selection

Support explicit layout selection per page. If the user requests a layout, preserve it unless it breaks reading order or text readability. If no layout is requested, choose the simplest layout that serves the story beat.

### Print Page Layouts

- **single splash**: 1 full-page panel for impact, reveal, establishing image, or emotional silence.
- **hero plus insets**: 1 dominant panel with 2-4 small inset reaction/detail panels.
- **two-tier cinematic**: 2 wide horizontal panels for slow drama, landscape, pursuit, or contrast.
- **three-strip page**: 3 stacked horizontal panels for clean cause-and-effect or quiet timing.
- **four-panel grid**: 2x2 grid for balanced dialogue, comparison, or controlled comedy.
- **six-panel grid**: 2x3 or 3x2 grid for standard scene progression.
- **nine-panel grid**: 3x3 grid for rhythm, repetition, investigation, comedy timing, or pressure.
- **vertical ladder**: tall narrow panels stepping down the page for falling, chasing, tension, or fast cuts.
- **diagonal action page**: angled panels and broken gutters for combat, panic, impact, or speed.
- **bleed impact page**: one panel bleeds to the page edge with smaller supporting panels.
- **conversation stack**: alternating medium shots and close-ups with clear balloon lanes.
- **reveal page-turn**: smaller buildup panels ending in a large final reveal panel.
- **spread layout**: two-page composition; use only when output format supports facing pages.

### 4-Koma Layouts

- **classic vertical 4-koma**: four equal stacked panels: setup, development, turn, punchline.
- **horizontal 4-koma**: four equal side-by-side panels for social/web use.
- **reaction 4-koma**: repeated camera with expression escalation.

### Webtoon Layouts

- **scroll stack**: sequential panels separated by vertical breathing room.
- **long descent**: tall vertical movement or emotional fall across multiple screen heights.
- **delayed reveal**: whitespace gap before a reveal panel.
- **chat rhythm**: alternating character panels with clean balloon lanes.
- **action cascade**: staggered panels, motion effects, and vertical impact beats.

### Layout Selection Rules

- Use splash or bleed layouts sparingly; they cost story space.
- Use grids when clarity, comedy rhythm, or repeated comparison matters.
- Use diagonal/broken layouts only when the scene is chaotic or kinetic.
- Use conversation stacks when dialogue must remain readable.
- For RTL manga, define the layout in right-to-left reading order. For LTR comics, define it left-to-right. For webtoon, define it top-to-bottom.
- If a page has heavy dialogue, reduce panel count or choose a layout with obvious balloon lanes.
- Name the selected layout in the script, `name` storyboard, and page prompt packet.

## Panel Language

Every panel description should specify:

- Shot scale: establishing, full body, medium, close-up, extreme close-up, insert.
- Camera angle: eye-level, low angle, high angle, overhead, dutch angle, profile, over-the-shoulder.
- Acting: expression, gesture, posture, tension, eye direction.
- Background density: blank emphasis, speed lines, symbolic background, simple setting, detailed setting.
- Text load: none, short line, dialogue exchange, caption, SFX.
- Layout role: dominant panel, inset, reaction beat, transition panel, reveal panel, or rhythm panel.

Use close-ups for emotional beats, inserts for clues/props, and wide panels for orientation or impact.

## Reading Direction

Lock the direction before storyboarding:

- **Japanese manga RTL**: right-to-left pages and panels. Balloon order follows right-to-left inside each panel.
- **Western comic LTR**: left-to-right pages and panels. Balloon order follows left-to-right inside each panel.
- **Webtoon vertical**: top-to-bottom scroll. Use vertical spacing for timing, silence, drops, and reveals.

Never place focal points so the eye-flow fights the selected direction.

## `Name` Storyboard Contents

For each page or webtoon beat, include:

- page/beat number
- selected layout template or custom layout
- panel count
- panel layout in reading order
- relative panel sizes
- focal point per panel
- exact dialogue, captions, and SFX
- balloon/caption/SFX placement
- background/tone notes
- transition from the previous page/beat

## Completion Checks

- [ ] The script has one entry per requested page or scroll beat.
- [ ] The `name` storyboard matches the script page/beat count.
- [ ] Every page has a clear story purpose.
- [ ] Panel count and density match the beat's emotional speed.
- [ ] Reading order is explicit and consistent.
- [ ] Text volume leaves room for art.
- [ ] Page turns, reveals, or scroll timing are planned.
